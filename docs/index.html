Python GUI Password Manager





What is a Password Manager?


A password manager is an application created to store and manage online credentials.


About this demo


This Python application is a Desktop-based complex password generator and manager which stores your credentials locally on your machine. The GUI was created with the Tkinter GUI toolkit, the complex password generator randomly creates passwords with letters, numbers, upper and lower characters and saves the new password to your clipboard with Pyperclip* and to a generated JSON local storage file.
